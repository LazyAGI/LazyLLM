# LazyLLM 代码模板

本目录包含 LazyLLM 的常用代码模板，可以直接使用或根据需求修改。

## 模板列表

### 基础模板

| 文件 | 功能 | 使用场景 |
|------|------|---------|
| `basic_chatbot.py` | 基础对话机器人 | 简单的对话应用、多轮对话 |

### RAG 模板

| 文件 | 功能 | 使用场景 |
|------|------|---------|
| `basic_rag.py` | 基础 RAG 系统 | 知识库问答、文档检索 |

### Agent 模板

| 文件 | 功能 | 使用场景 |
|------|------|---------|
| `basic_agent.py` | 智能体工具调用 | 代码生成、任务执行、工具编排 |

### Flow 模板

| 文件 | 功能 | 使用场景 |
|------|------|---------|
| `basic_flow.py` | 数据流编排 | Pipeline、Parallel、复杂流程处理 |

### 微调模板

| 文件 | 功能 | 使用场景 |
|------|------|---------|
| `basic_finetune.py` | 模型微调 | 模型训练、参数优化 |

### 多模态模板

| 文件 | 功能 | 使用场景 |
|------|------|---------|
| `basic_multimodal.py` | 多模态智能体 | 语音识别、图片问答、画图、音乐生成、TTS |

### 高级功能模板

| 文件 | 功能 | 使用场景 |
|------|------|---------|
| `output_formatter.py` | 输出格式化 | JSON 解析、自定义格式化 |
| `stream_output.py` | 流式输出 | 实时输出、流式配置 |
