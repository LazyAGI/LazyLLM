# Store 与数据源参考（本地说明）

各 Store（OceanBase、SenseCore、OpenSearch、ElasticSearch、Chroma、Milvus 等）的详细连接字段与配置项以对应产品文档为准。  
本 skill 内仅做用法摘要；完整字段说明需在联网时查阅各产品官方文档。

- **OceanBase**：连接与字段见 [oceanbase官方文档](https://www.oceanbase.com/docs/oceanbase-database-cn)。
- **SenseCore**：存储与配置见 [SenseCore官方文档](https://www.sensecore.cn/help#storage)。
- **OpenSearch**: 支持的字段详细参考[opensearch官方文档](https://opensearch-project.github.io/opensearch-py/api-ref/clients/opensearch_client.html)
- **ElasticSearch**: 支持的字段详细参考[elasticsearch官方文档](https://www.elastic.co/docs/api/doc/elasticsearch/)
- **Chroma**: 支持的字段详细参考[chroma官方文档](https://docs.trychroma.com/)
- **Milvus**: 支持的字段详细参考 [milvus官方文档](https://milvus.io/docs/v2.3.0/index.md)

本目录下 [store.md](store.md) 提供基础用法与示例；扩展配置请参考项目主 docs 或上述产品文档（需联网）。
